关于 CarWiki

项目介绍

CarWiki 是一个开源的 1/64 比例合金车模数据聚合项目。我们通过自动化爬虫从各大品牌官网采集产品数据，并以美观直观的方式呈现给车模爱好者。

数据来源

所有数据均来自各品牌官方网站的公开信息。爬虫遵循 robots.txt 协议，并设置了合理的请求间隔以减少对源站的负载。

技术架构

  • 爬虫：Python + BeautifulSoup + Requests
  • 前端：原生 HTML/CSS/JavaScript
  • 部署：GitHub Pages（静态托管）
  • 自动化：GitHub Actions 定时触发爬虫
